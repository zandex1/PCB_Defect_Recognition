1. Создание проекта

В верхнем меню нажать "файл", выбрать пункт создать проект.
Появится окно в котором, будут поля для ввода "названия проекта" и "места расположния", причем для ввода места расположения 
можно будет открыть файловый менеджер. По окончании выбора создает папку с проектом в указанном месте, с файлом информации о проекте.
(QMenuBar,QDir, QFileSystemModel, pushButton, TextEdit) 

2. Открытие проекта

В верхнем меню нажать "файл", выбрать пункт открыть проект.
Появится окно в котором, нужно будет указать путь к файлу (*.(Придумать тип)).
(QMenuBar, QDir QFileSystemModel, pushButton, TextEdit)

4. Настройки камеры
Меню настройки камеры имеет два раздела: Управление камерой и Усилитель видеопроцессора.
В управлении камерой можно: выбрать используемую камеру, настроить масштаб, фокус, выдержку, диафрагму.
В усилители видеопроцессора можно настроить: яркость, контрастность, оттенок, насыщенность, четкость, гамму, баланс белого, усиление.
(QMenuBar, QTabWijet, OpenCV VideoCaptureProperties, pushButton, slider, TextEdit, checkBox)

5. Сканирование
Имеет кнопки для запуска сканирования, сегментизации платы, выбор что сканируется (референс или инспектируемая).
Также имеет два вывода изображения: текущее положение камеры с перекрестием по центру и после склейки.
(QTabWijet, OpenCV, pushButton, slider, TextEdit, checkBox)

6. Распознавание
На изображении рефренсе распознаются SMD компоненты с указанием их типа, 
потом по расположению данных компонентов идет поиск по инспектируемой плате.
